Set up virustotal-scan.sh

In order to use virustotal-scan.sh, you need to retrieve your own, personal API key for virustotal.com.

  1. Register at https://www.virustotal.com/gui/join-us
  2. After registering and logging into Virustotal, go to the Virustotal start page, click on your user name/icon on the upper right and select API key. This will bring you to the API key section of in profile.
    From there, copy your API key.
  3. Open {SCRIPT_PATH} in a text editor and replace '<YOUR API KEY>' with your API key. Don't forget to save! Now you can start using virustotal-scan.sh by right clicking on a file within Nautilus. You will find it in the context menu under 'Scripts'.
